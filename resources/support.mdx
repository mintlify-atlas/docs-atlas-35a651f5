## Problemas Técnicos Comunes

### Solución de Problemas

Antes de contactar soporte, intenta estas soluciones comunes:

<AccordionGroup>
  <Accordion title="La página no carga correctamente">
    **Posibles soluciones:**

    1. **Limpia la caché del navegador**:
       - Chrome: `Ctrl + Shift + Delete` (Windows) o `Cmd + Shift + Delete` (Mac)
       - Firefox: `Ctrl + Shift + Delete`
       - Safari: `Cmd + Option + E`

    2. **Verifica tu conexión a internet**:
       - Intenta cargar otros sitios web
       - Reinicia tu router si es necesario

    3. **Actualiza tu navegador**:
       - Asegúrate de usar la última versión
       - Prueba con un navegador diferente

    4. **Desactiva extensiones**:
       - Algunas extensiones pueden interferir con la página
       - Prueba en modo incógnito/privado

    Si el problema persiste, repórtalo en [GitHub Issues](https://github.com/alexander10pb/UniJac/issues).
  </Accordion>

  <Accordion title="No puedo subir imágenes en los reportes">
    **Verifica lo siguiente:**

    1. **Tamaño del archivo**:
       - Las imágenes deben ser menores a 5MB
       - Comprime imágenes grandes antes de subirlas

    2. **Formato de imagen**:
       - Formatos permitidos: JPG, PNG, GIF, WebP
       - No se permiten archivos SVG por seguridad

    3. **Permisos del navegador**:
       - Asegúrate de permitir acceso a archivos
       - En móvil, permite acceso a la cámara/galería

    4. **Conexión a internet**:
       - Las subidas requieren conexión estable
       - Evita subir con conexión muy lenta

    **Tip**: Intenta subir la imagen nuevamente o prueba con una imagen diferente.
  </Accordion>

  <Accordion title="No recibo notificaciones">
    **Verifica tu configuración**:

    1. **Notificaciones del navegador**:
       - Asegúrate de haber permitido notificaciones para UniJAC
       - Revisa la configuración de notificaciones de tu navegador

    2. **Configuración de cuenta**:
       - Inicia sesión en tu cuenta
       - Ve a "Configuración" > "Notificaciones"
       - Activa las notificaciones que desees recibir

    3. **Correo electrónico**:
       - Verifica tu carpeta de spam/correo no deseado
       - Añade el correo de UniJAC a tu lista de contactos

    4. **Conexión activa**:
       - Mantén la página abierta para notificaciones en tiempo real
       - Las notificaciones por correo pueden tardar unos minutos
  </Accordion>

  <Accordion title="Olvidé mi contraseña">
    **Para recuperar tu contraseña**:

    1. Haz clic en "Iniciar sesión" en la página principal
    2. Selecciona "¿Olvidaste tu contraseña?"
    3. Ingresa tu correo electrónico registrado
    4. Revisa tu correo para el enlace de recuperación
    5. Sigue las instrucciones para crear una nueva contraseña

    **Consejos de seguridad**:
    - Usa una contraseña segura con al menos 8 caracteres
    - Combina letras mayúsculas, minúsculas, números y símbolos
    - No uses la misma contraseña en múltiples sitios
    - Considera usar un gestor de contraseñas

    Si no recibes el correo de recuperación, verifica tu carpeta de spam o contacta al soporte.
  </Accordion>

  <Accordion title="La plataforma está lenta">
    **Optimiza tu experiencia**:

    1. **Cierra pestañas innecesarias**:
       - Libera memoria del navegador
       - Cierra aplicaciones que consuman recursos

    2. **Verifica tu conexión**:
       - Prueba la velocidad de tu internet
       - Conéctate a una red más estable si es posible

    3. **Actualiza tu navegador**:
       - Usa la versión más reciente
       - Considera usar un navegador más ligero

    4. **Dispositivo**:
       - Cierra aplicaciones en segundo plano
       - Reinicia tu dispositivo si es necesario

    Si la lentitud persiste para todos los usuarios, podría ser un problema del servidor. Repórtalo en GitHub.
  </Accordion>
</AccordionGroup>

## Reportar Bugs

Si encuentras un error en la plataforma:

<Steps>
  <Step title="Verifica que sea un bug">
    - ¿El problema es reproducible?
    - ¿Ocurre en diferentes navegadores?
    - ¿Ya revisaste la sección de problemas comunes?
  </Step>

  <Step title="Revisa issues existentes">
    Visita [GitHub Issues](https://github.com/alexander10pb/UniJac/issues) para ver si ya fue reportado.
  </Step>

  <Step title="Crea un nuevo issue">
    Si es un bug nuevo:
    - Usa un título descriptivo
    - Describe los pasos para reproducirlo
    - Incluye capturas de pantalla
    - Especifica tu navegador y sistema operativo
  </Step>
</Steps>

<Card title="Reportar Bug" icon="bug" href="https://github.com/alexander10pb/UniJac/issues/new">
  Crea un nuevo issue en GitHub para reportar un bug
</Card>
