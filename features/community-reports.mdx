## Overview

UniJAC enables residents to report issues and share information across six key categories. Each report helps build a comprehensive view of neighborhood conditions and facilitates community action.

## Report Categories

The platform supports six report categories visible throughout the interface:

### Report Structure

Each report includes:

1. **Title** - Brief description of the issue
2. **Timestamp** - When the report was submitted
3. **Author** - Username or "Vecino anónimo"
4. **Category** - One of six categories
5. **Description** - Detailed explanation
6. **Evidence** - Optional attachments ("Adjunto")
7. **Severity** - Moderado, Alto, Crítico
8. **Status** - Pendiente, En progreso, Resuelto

## After Submission

### Report Visibility

Submitted reports appear in multiple locations:

<Steps>
  <Step title="Dashboard Summary">
    Report counts update in real-time on the neighborhood dashboard
    
    ```html
    <article class="card">
      <p>Daños</p>
      <span class="badge">6</span>
    </article>
    ```
  </Step>
  
  <Step title="Category Pages">
    Reports are listed on dedicated category pages with filtering options
    
    ```html

        <option value="60">Últimos 60 días</option>
    ```

  </Step>
  
  <Step title="Community Feed">

  </Step>
</Steps>

## Best Practices

<AccordionGroup>
  <Accordion title="Writing Effective Reports">
    - Use clear, specific titles
    - Include exact location details
    - Provide context and impact
    - Attach photos when possible
    - Update status as situations change
  </Accordion>
  
  <Accordion title="Choosing Report Types">
    - Select the most specific category
    - Use Infrastructure for physical damage
    - Use Sanitation for cleanliness issues
    - Use Security for safety concerns
    - Use Public Services for utility failures
  </Accordion>
  
  <Accordion title="Anonymity Considerations">
    - Anonymous reports still create visibility
    - Identified reports enable follow-up
    - Consider community impact vs privacy
    - JAC members can view all submission details
  </Accordion>
</AccordionGroup>
