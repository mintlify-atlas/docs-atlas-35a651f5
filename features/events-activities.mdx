## Overview

UniJAC's Events & Activities feature helps residents stay informed about community gatherings, activities, and organized events. From fitness classes to vaccination drives, discover everything happening in your neighborhood.

## Participating in Events

### RSVP System

Events feature an attendance confirmation button:

```html RSVP Button
<button type="button" aria-disabled="true">
  Asistiré
</button>
```

<Steps>
  <Step title="Browse Events">
    Navigate to the Events & Activities page or view previews on the dashboard
  </Step>
  
  <Step title="Review Details">
    Check date, time, location, and organizer information
  </Step>
  
  <Step title="Confirm Attendance">
    Click the "Asistiré" (I will attend) button to register
  </Step>
  
  <Step title="Receive Updates">
    Get notifications about event changes or cancellations
  </Step>
</Steps>

<Warning>
The RSVP functionality requires user authentication. Make sure you're logged in before attempting to confirm attendance.
</Warning>

## Best Practices

<Tip>
**For Event Organizers:**
- Post events at least 1 week in advance
- Include clear location and time details
- Provide contact information for questions
- Update event status if changes occur
- Add relevant images to increase engagement
</Tip>

<Tip>
**For Participants:**
- RSVP early to help organizers plan
- Check event details 24 hours before
- Arrive on time to the specified location
- Share events with neighbors who might be interested
- Provide feedback after attending
</Tip>
