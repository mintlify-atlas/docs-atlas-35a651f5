## Overview

The Neighborhood Dashboard is the central hub for all community information in UniJAC. It provides a comprehensive view of reports, events, schedules, and neighborhood activity through an intuitive layout accessible at `neighborhoods.html`.

## Dashboard Layout

## Responsive Design

The dashboard adapts to different screen sizes:

<Tabs>
  <Tab title="Desktop">
    - Left sidebar always visible
    - Three-column layout
    - Full navigation in header
    - Expanded report cards
  </Tab>
  <Tab title="Tablet">
    - Collapsible sidebar
    - Two-column layout
    - Hamburger menu for navigation
    - Condensed cards
  </Tab>
  <Tab title="Mobile">
    - Hidden sidebar (accessible via button)
    - Single-column layout
    - Bottom navigation possible
    - Stacked cards
  </Tab>
</Tabs>
