This guide will walk you through creating your UniJAC account and help you get to your first meaningful action in the platform.

## Prerequisites

Before you begin, make sure you have:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Your neighborhood information (localidad and barrio in Bogotá)
- An email address for account registration

## Step 1: Create your account

UniJAC uses a two-step registration process to ensure you're connected to the right neighborhood community.

<Steps>
  <Step title="Navigate to registration">
    Visit the UniJAC platform and click the "Registrarse" button in the header, or navigate directly to the registration page using the hash route:
    
    ```javascript
    window.location.hash = '#/register';
    ```
    
    The platform uses hash-based routing handled by the routes.js file:
    
    ```javascript
    case "#/register":
      await loadView("pages/register.html");
      if (typeof initRegister === "function") initRegister();
      break;
    ```
  </Step>
  
  <Step title="Complete Step 1 - Location and basic info">
    Fill in your neighborhood details and personal information:
    
    - **Localidad**: Select your locality from the dropdown
    - **Barrio**: Select your specific neighborhood (enabled after selecting localidad)
    - **Nombre**: Enter your first name
    - **Apellido**: Enter your last name
    
    <CodeGroup>
    ```html Step 1 Form Fields
    <div class="form-group">
      <label for="localidad">Localidad</label>
      <select name="localidad" id="localidad" aria-live="polite" required>
        <option selected disabled hidden>Selecciona localidad</option>
      </select>
    </div>

    <div class="form-group">
      <label for="barrios">Barrio</label>
      <select name="barrios" id="barrios" aria-live="polite" required disabled>
        <option selected disabled hidden>Selecciona barrio</option>
      </select>
    </div>

    <div class="form-group">
      <label for="nombre">Nombre</label>
      <input type="text" name="nombre" id="nombre" required />
    </div>

    <div class="form-group">
      <label for="apellido">Apellido</label>
      <input type="text" name="apellido" id="apellido" required />
    </div>
    ```
    </CodeGroup>
    
    <Info>
      By continuing, you accept the use of cookies and privacy policy for UniJAC account creation.
    </Info>
    
    Click "Continuar" to proceed to Step 2.
  </Step>
  
  <Step title="Complete Step 2 - Account credentials">
    Set up your login credentials:
    
    - **Correo electrónico**: Enter your email address
    - **Contraseña**: Create a secure password
    
    ```html
    <div class="form-group">
      <label for="correo">Correo electrónico</label>
      <input type="email" name="correo" id="correo" required />
    </div>

    <div class="form-group">
      <label for="password">Contraseña</label>
      <input type="password" name="password" id="password" required />
    </div>
    ```
    
    Click "Crear cuenta" to complete registration.
  </Step>
</Steps>

<Note>
  The registration form shows "Paso 1 de 2" and "Paso 2 de 2" to help you track your progress through the signup process.
</Note>

## Step 2: Select your neighborhood

After creating your account, you'll see a welcome modal to confirm your neighborhood selection.

<Steps>
  <Step title="Welcome modal appears">
    When you first access the neighborhoods dashboard, you'll see a welcome message:
    
    ```html
    <div class="modal-content">
      <div class="modal-header">
        <h2 class="title-modal">¡Te damos la bienvenida! 🎉</h2>
      </div>
      <div class="modal-body">
        <p>Antes de continuar, necesitamos encontrar tu barrio.</p>
      </div>
    </div>
    ```
  </Step>
  
  <Step title="Search for your barrio">
    Use the search input to find and select your neighborhood:
    
    ```html
    <input
      id="buscar-barrio"
      type="search"
      autocomplete="off"
      placeholder="Escribe tu barrio"
      list="barrios-lista"
      class="input-search"
    />
    ```
    
    The datalist provides autocomplete suggestions as you type.
  </Step>
  
  <Step title="Confirm and continue">
    Click the "Continuar" button to access your neighborhood dashboard.
  </Step>
</Steps>

## Navigation patterns

UniJAC uses hash-based routing for seamless navigation. Understanding this pattern helps you navigate the platform:

```javascript
// Home page
window.location.hash = '#/home';

// Registration
window.location.hash = '#/register';

// Login
window.location.hash = '#/login';

// About page
window.location.hash = '#/about';

// Contact
window.location.hash = '#/contact';
```

The routing system automatically handles page transitions:

```javascript
async function hashRoutes() {
  window.scrollTo(0, 0);
  const route = location.hash || "#/home";
  const isHome = route === "#/" || route === "#/home";
  
  // Header styling adjusts based on route
  if (header) header.classList.toggle("light-theme", !isHome);
  
  // Register button visibility
  if (btnRegister) {
    btnRegister.style.display = route === "#/register" ? "none" : "flex";
  }
  
  // Route handling
  switch (route) {
    case "#/home":
      if (main) main.innerHTML = homeHTML;
      initHome();
      break;
    case "#/register":
      await loadView("pages/register.html");
      if (typeof initRegister === "function") initRegister();
      break;
    // ... other routes
  }
}

window.addEventListener("hashchange", hashRoutes);
```

<Tip>
  The platform automatically scrolls to the top of the page on route changes for a better user experience.
</Tip>

## Accessibility features

UniJAC includes built-in accessibility controls. Click the accessibility button (♿) in the bottom right to access:

- **Aumentar fuente**: Increase font size
- **Reducir fuente**: Decrease font size
- **Contraste**: Toggle high contrast mode

These features help ensure the platform is usable for everyone in your community.
